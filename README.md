Modern Login Page

A sleek, animated, and fully responsive login & registration interface built with **HTML**, **CSS**, and **JavaScript**.  
This project features a smooth sliding panel effect, social icon buttons, and a modern gradient UI.

---

Features

Toggle animation between **Sign In** and **Sign Up**
Clean and modern UI with smooth transitions
Fully responsive design for all screen sizes
Includes social login button placeholders
Lightweight: pure HTML, CSS, and JavaScript (no frameworks required)

---

Project Structure

index.html # Main HTML structure
style.css # Styling and animations
script.js # Toggle logic for the forms


---
